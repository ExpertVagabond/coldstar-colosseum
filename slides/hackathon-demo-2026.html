01 — HARDWARE WALLETS

What is a hardware wallet?

A device that keeps your private keys offline — away from hackers, malware, and phishing attacks

🔒

Private Keys

The password to your crypto. Whoever has it controls your funds. It must never touch the internet.

💻

Software Wallets

Phantom, MetaMask — convenient but your keys live in your browser. One exploit = drained.

🛡

Hardware Wallets

Keys stay on a separate device. Transactions are signed offline. The gold standard of security.

03 — INSTALLATION

Get started in 3 steps

01
Clone & Install
git clone the repo.
Run the install script.
Python + Rust compile in ~2 min.
02
Flash Any USB
Plug in a USB drive — any brand, any size.
The TUI detects it automatically.
One command encrypts & provisions.
03
Create Wallet
Generate a Solana keypair offline.
Keys encrypted with AES-256-GCM.
Locked in RAM with mlock. Done.
